### How to start?
---

1. 啟動 apache and mysql server in XAMPP
2. clone this repo to path: xampp/htdocs
3. 開啟 phpMyadmin 匯入 ```FERNITURE_DB.sql``` 
4. Browser 輸入 http://127.0.0.1/113DBMS/public/?url=home
